 Ce dépôt héberge le code source utilisé durant le TP sur l\textquotesingle{}outil Doxygen. Il contient deux classes codées en C++ \+:


\begin{DoxyItemize}
\item une classe \doxylink{class_individu}{Individu} qui permet de représenter des individus définis par un nom et un prénom ;
\item une classe \doxylink{class_voiture}{Voiture} qui permet de représenter des voitures définies par une marque et une plaque d\textquotesingle{}immatriculation.
\end{DoxyItemize}

Un \doxylink{class_individu}{Individu} peut conduire au plus une \doxylink{class_voiture}{Voiture} et une \doxylink{class_voiture}{Voiture} peut être conduite par au plus un \doxylink{class_individu}{Individu}. Le code présent sur ce dépôt montre comment implanter correctement cette relation de manière bidirectionnelle tout en maintenant une cohérence lorsqu\textquotesingle{}on ajoute ou supprime une relation entre un \doxylink{class_individu}{Individu} et une \doxylink{class_voiture}{Voiture}.
